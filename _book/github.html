Configurando o Git e Github

Capítulo 3 Github

O Github é um serviço de hospedagem para projetos que utilizam o controle de versão Git. Nessa plataforma podemos encontrar todos as commits dos projetos que estão lá hospedados,também acompanhar o desenvolvimento de projetos, quantas pessoas estão envolvidas com o projeto e etc.

3.1 Criando sua conta

Para criar sua conta no Github é bem simples e rápido. Basta apenas acessar o link: www.github.com e clicar em “Sign up”.
